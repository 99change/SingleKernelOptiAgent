GPU Kernel Optimization Report
============================================================

Baseline time    : 0.44 ms
Optimized time   : 0.40 ms
Speedup          : 10.9%

Bottlenecks identified:
  - Memory bandwidth saturation due to low arithmetic intensity
  - Suboptimal memory transaction granularity using scalar loads/stores

Strategies applied:
  - Use vectorized memory access (float4) to reduce instruction overhead and improve bandwidth utilization
  - Apply read-only cache hints (__ldg) for input arrays to utilize read-only data cache
